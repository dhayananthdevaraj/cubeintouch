<p><strong><u>Problem Statement</u></strong></p>

<p>A railway reservation system is being developed to automate ticket fare calculation for passengers. When a passenger approaches the booking counter, the staff enters the <b>travel class</b>, the <b>distance in kilometres</b>, and whether the passenger is a <b>senior citizen</b> to determine the final ticket fare.</p>
 
<p>The railway board has fixed the following base fare rates per kilometre for each travel class:</p>
<ul>
<li>Class <b>1</b> (Sleeper) — ₹0.50 per km</li>
<li>Class <b>2</b> (AC 3-Tier) — ₹1.20 per km</li>
<li>Class <b>3</b> (AC 1-Tier) — ₹2.50 per km</li>
</ul>
 
<p>Senior citizens (age 60 and above) are entitled to a <b>40% concession</b> on the base fare as per government policy. Additionally, a <b>minimum fare of ₹50</b> is applicable for all tickets regardless of distance or concession. The system uses a user-defined function <b>trainFare()</b> that takes the class, distance, and senior citizen flag (1 for yes, 0 for no), computes the fare, and returns it as an integer (floor value).</p>
 
<p>The program reads the inputs, calls the function, and displays the final ticket fare. However, the provided code contains certain issues and may not execute correctly. Your task is to identify and fix the bugs so that the program runs successfully.</p>

The first line contains three space-separated integers — Class (1, 2, or 3), Distance (in km), and Senior Citizen flag (1 or 0).
 
OUTPUT_FORMAT:
Print the following single line:
Ticket Fare: Rs.<amount>

The given testcases falls under the following constraints:
1 ≤ Distance ≤ 2000
Class is 1, 2, or 3
Senior Citizen flag is 0 or 1
